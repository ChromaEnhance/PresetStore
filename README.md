## 💻 The Desktop Client

The PresetStore Desktop Application redefines asset management. Built for performance, it completely automates the installation process so creators can focus on production.

* **Zero-Configuration Deployment:** Installs presets instantly without manual file routing.
* **Intelligent Environment Detection:** Automatically identifies compatible host applications (e.g., After Effects) and locates the most recent installed versions on your system.
* **Complete Lifecycle Management:** Features a built-in, clean installer and uninstaller for total control over your local library.

---

## ⚙️ Technical Architecture

PresetStore operates on a modern, high-performance stack designed for scalability and speed.

* **Backend Infrastructure:** Node.js, Hapi.js, Knex, PostgreSQL
* **Asset Storage:** RustFS (S3-compatible scalable storage)
* **Authentication:** OAuth 2.0 (Google & Discord Integration)
* **Desktop Application:** WPF (.NET 10) featuring a modern Fluent UI framework (WPF UI / Lepo.co)
* **Web Environment:** HTML / CSS / JavaScript with a seamless Fluent design language
